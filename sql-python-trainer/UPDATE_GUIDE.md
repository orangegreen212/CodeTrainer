## 🚀 Как обновить проект на GitHub

### Шаг 1: Замените все файлы

1. **Удалите старые файлы** из вашего репозитория (или переименуйте папку)
2. **Скачайте** все новые файлы из этой папки
3. **Загрузите** их в GitHub репозиторий

### Шаг 2: Загрузите на GitHub

```bash
# Перейдите в папку с новыми файлами
cd путь/к/новым/файлам

# Инициализируйте git (если нужно)
git init

# Добавьте remote (ваш существующий репозиторий)
git remote add origin https://github.com/orangegreen212/CodeTrainer.git

# Добавьте все файлы
git add .

# Закоммитьте
git commit -m "Update: New schema, 85 tasks, improved design, Mistral validation"

# Отправьте (возможно потребуется force push)
git push -f origin main
```

### Шаг 3: Настройте Mistral API (опционально)

Если хотите проверку решений через AI:

1. Зарегистрируйтесь на https://mistral.ai
2. Получите бесплатный API ключ
3. В Render Dashboard → Environment → Add:
   - Key: `MISTRAL_API_KEY`
   - Value: ваш ключ
4. Save Changes

**Без этого все будет работать, кроме кнопки "Проверить решение"**

## 🐛 Если что-то не работает

**База данных не обновилась:**
1. Render Dashboard → PostgreSQL
2. Delete Database
3. Create Database заново
4. Обновите DATABASE_URL в Web Service

**Ошибка Python 3.14:**
1. Render Dashboard → Environment
2. Add: `PYTHON_VERSION = 3.12.8`

**Старые файлы остались:**
1. Убедитесь, что сделали `git push -f`
2. Проверьте Root Directory в Render Settings
